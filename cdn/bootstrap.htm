Bootstrap is a mobile optimized .css file package that helps with containers and with easing the process of alerts, menus, image carousels, etc.

How to add bootstrap to my code?

Copy and paste the code into the head of your HTML / PHP file.


<link rel="stylesheet" href="https://sans-skelly.github.io/cdn/upload_files/bootstrap.min.css">
  <script src="https://sans-skelly.github.io/cdn/upload_files/bootstrap.min.js"></script>
